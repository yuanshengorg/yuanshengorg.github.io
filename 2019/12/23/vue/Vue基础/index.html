    什么是指令？

    指令的本质就是自定义属性；以 v- 开头；

    闪动：html页面被刷新时，会先显示 大括号 ，再显示数据；

    v-clack：先通过样式隐藏内容，然后在内存中进行值的替换，替换完成之后再显示最终的结果；

    <style>
    [v-cloak] {
        display: none;
    }
</style>

<body>
    <div v-cloak> {{msg}} </div>
</body>

    v-on 处理事件

    v-on:click='num++'
@click='num++'

    点击加1：

    <div id="app" v-cloak>
    <div>{{num}}</div>
    <button v-on:click='num++'>点击加1</button>
    <button @click='num++'>点击加1</button>
</div>
<script src="./js/vue.js"></script>
<script>
    var vm = new Vue({
        el: '#app',
        data: {
            num: 0
        }
    })
</script>

    使用函数的方式：

    <div id="app" v-cloak>
    <div>{{num}}</div>
    <!-- 直接绑定函数名称 handle -->
    <button v-on:click='handle'>点击加1</button>
    <!-- 调用函数 handle() -->
    <button v-on:click='handle()'>点击加1</button>
</div>
<script src="./js/vue.js"></script>
<script>
    var vm = new Vue({
        el: '#app',
        data: {
            num: 0
        },
        methods: {
            handle: function () {
                // 这里的this是Vue的实例对象
                console.log(this === vm)
                this.num++;
            }
        }
    })
</script>

    事件函数调用

    事件绑定-参数传递

    如果事件直接绑定函数名称，默认会传递事件对象作为事件函数的第一个参数；

    如果事件绑定函数调用，那么事件对象必须作为最后一个参数显示传递，名称为：event；

    <div id="app" v-cloak>
    <div>{{num}}</div>
    <!-- 如果事件直接绑定函数名handle，默认会传递事件对象作为函数的第一个参数 -->
    <button v-on:click='handle1'>点击1</button>
    <!-- 如果事件绑定函数调用handle()，那么事件对象必须作为最后一个参数显示传递，并且事件对象的名称必须是固定的$event -->
    <button v-on:click='handle2(123, 456, $event)'>点击2</button>
</div>
<script src="./js/vue.js"></script>
<script>
    var vm = new Vue({
        el: '#app',
        data: {
            num: 0
        },
        methods: {
            handle1: function (event) {
                console.log(event.target.innerHTML);
            },
            handle2: function (p1, p2, event) {
                console.log(p1, p2);
                console.log(event.target.innerHTML);
            }
        }
    })
</script>

    事件修饰符

    .stop阻止冒泡

    <a v-on:click.stop='handle'>跳转</a>

    .prevent 阻止默认行为

    <a v-on:click.prvevnet='handle'>跳转</a>

    例：

    <div id="app" v-cloak>
    <div>{{num}}</div>
    <div v-on:click='handle1'>
        <button v-on:click='handle2'>点击不加1</button>
        <button v-on:click.stop='handle2'>点击不加1</button>
    </div>
    <a href="http://www.halfbamboo.com" v-on:click='handle3'>不跳转</a>
    <a href="http://www.halfbamboo.com" v-on:click.prevent='handle3'>不跳转</a>
</div>
<script src="./js/vue.js"></script>
<script>
    var vm = new Vue({
        el: '#app',
        data: {
            num: 0
        },
        methods: {
            handle1: function () {
                this.num++;
            },
            handle2: function (event) {
                // 使用原生方式阻止冒泡
                // 点击button，不能让num加1，和.stop一样
                event.stopPropagation();
            },
            handle3: function (envent) {
                // 使用原生方式阻止默认行为
                // 阻止跳转，和.prevent一样
                event.preventDefault();
            }
        }
    })
</script>

    <div>
        <!-- 在输入完密码后直接按回车就可以提交 -->
        密码：<input type="text" v-model='pwd' v-on:keyup.enter='handleSubmit'>

    <div>
        <input type="button" v-on:click='handleSubmit' value="提交">
    </div>

<script src="./js/vue.js"></script>
<script>

        el: '#app',
        data: {
            uname: '',
            pwd: ''
        },
        methods: {
            handleSubmit: function () {
                console.log(this.uname);
                console.log(this.pwd);
            },
            clearContent: function () {
                // 按del时清空用户名
                this.uname = '';
            }
        }
    })
</script>

    v-model 的底层实现原理：

    <div id="app">
    <div>{{msg}}</div>
    <!-- 修改input不会改变div的值 只是将值msg绑定给input的value属性 -->
    <input type="text" v-bind:value='msg'>
    <!-- 修改input改变div的值 -->
    <input type="text" v-bind:value='msg' v-on:input='handle'>
    <!-- 与上面相同，不用函数 -->
    <input type="text" v-bind:value='msg' v-on:input='msg=$event.target.value'>
    <!-- 和v-model相同 -->
    <input type="text" v-model='msg'>
</div>
<script src="./js/vue.js"></script>
<script>
    var vm = new Vue({
        el: '#app',
        data: {
            msg: 'hi, Vue'
        },
        methods: {
            handle: function (event) {
                // 使用输入域中的最新数据覆盖原来的数据
                this.msg = event.target.value;
            }
        }
    })
</script>
